# MarioGame

A 2D platformer game built with Unity and C# featuring a Ninja Frog protagonist. Players navigate through challenging levels, avoid obstacles, defeat enemies, collect items, and reach the end of each stage.

## Features

* Smooth player movement and jumping mechanics
* Physics-based platforming
* Enemy and obstacle interactions
* Collectible items and score tracking
* Animated character using Unity Animator
* Level progression and game-over system
* Responsive collision detection

## Technologies Used

* Unity Game Engine
* C#
* Unity Physics System (Rigidbody2D, Colliders)
* Unity Animator
* Tilemap System

## Gameplay

Control the Ninja Frog as it traverses platforms, avoids hazards, and collects items throughout the level. The objective is to reach the destination while overcoming obstacles and challenges.

## Controls

* A / Left Arrow: Move Left
* D / Right Arrow: Move Right
* Space: Jump
